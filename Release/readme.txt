보건실 클리어 -> 교무실 열쇠 필요 확인 -> 도서관 좀비 정보 확인 ->
급식실 혈액 준비 -> 방송실 급식 방송 -> 급식실 문 잠금 -> 
자물쇠 제거 -> 세콤 무력화 -> 엔딩

혈액 팩 -> 점프 투 EMP -> EMP 장치 -> 교무실 열쇠 -> 핸드폰

교무실 열쇠 필요 확인 충족하면 1층에 갈 수 없게 됨
도서관 좀비 정보 확인 충족하면 방송실 이벤트 가능
급식실 문 잠금 충족 못하면 교무실에서 나오자마자 죽음

혈액팩 없으면 급식실이 있는 5층에 가면 죽음
정보 교과서 없으면 컴퓨터실에서 emp장치 못만듬
emp장치랑 교무실 열쇠 없으면 교무실 못들어감
핸드폰 없으면 1층에서 밖으로 탈출 못함
